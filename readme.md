# HFSS SMS Automation
## This is going to be a project for HFSS in order to be able to handle Automation rules using Twilio and Google Sheets

### dependencies used
[google sheets](https://www.npmjs.com/package/google-spreadsheet)
[express](https://www.npmjs.com/package/express)
[express-async-handler](https://www.npmjs.com/package/express-async-handler)
[twilio](https://www.npmjs.com/package/twilio)
[body-parser](https://www.npmjs.com/package/body-parser)
[googleapis](https://www.npmjs.com/package/googleapis)

### Oauth2 
credentials are located in .env file, when wanting to add a new spreadsheet to this web server make sure that you add the service email as an editor in the share settings of the document
